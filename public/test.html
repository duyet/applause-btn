Welcome to Applause Button

A self-hosted, privacy-focused applause button service for your website or blog. Unlike traditional "like" buttons, the applause button lets readers show varying levels of appreciation by clapping multiple times.

Try it out! 👇

Click and hold to give multiple claps

Each IP can clap once!

This prevents spam and abuse

💡 How to use

Add this code to your website to integrate the applause button:

<!-- Add to your HTML <head> --> <link rel="stylesheet" href="https://your-domain.com/public/applause-button.css" /> <script src="https://your-domain.com/public/applause-button.js"></script> <!-- Add where you want the button to appear --> <applause-button style="width: 58px; height: 58px" multiclap="true" api="https://your-domain.com" />

API Endpoints

This service provides the following REST API endpoints:

  • GET /health - Service health check
  • GET /get-claps - Get clap count for a URL
  • GET /get-clappers - Get clapper information
  • POST /get-multiple - Batch query multiple URLs
  • POST /update-claps - Add claps to a URL
